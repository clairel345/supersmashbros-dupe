# Super Smash Bros Dupe

A real-time 2 player fighting simulator built using Pygame. The game allows 2 players to compete on the same keyboard with responsive controls.

# Features 
- Replica of the Nintendo game Super Smash Bros, but with my own twist
- Character movement and attack depictions
- Collision detection and hit registration
- Health bars and win conditions
- Real-time game loop with smooth rendering

# Technologies
- Python
- Pygame

# Controls
- Player 1: WASD to move, SPACE to attack
- Player 2: Arrow keys to move, M to attack

# How To Run
- Clone the repository
- Install pygame 'pip install pygame'
- Run 'python3 performance_task.py'
